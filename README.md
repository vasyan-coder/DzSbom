# sbom-python-demo

Minimal Python project used to demonstrate:

- CycloneDX SBOM generation
- OWASP Dependency-Check scan
- OWASP Dependency-Track BOM ingestion
- Automation via GitLab CI

The dependency versions are intentionally pinned and include some older packages to produce findings.
